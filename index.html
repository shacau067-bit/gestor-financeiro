⚙️ Falta configurar o backend

Este app usa o Supabase como banco de dados real. Antes de usar, você precisa:

  1. Criar um projeto grátis em supabase.com
  2. Rodar o SQL de configuração no SQL Editor do Supabase
  3. Copiar a Project URL e a anon public key em Project Settings → API
  4. Editar este arquivo .html: procure por SUPABASE_URL e SUPABASE_ANON_KEY no início do <script> e cole seus valores no lugar de COLE_SUA_URL_AQUI e COLE_SUA_CHAVE_AQUI
  5. Salve o arquivo e abra de novo

Depois disso, qualquer pessoa com o arquivo (ou o link, se você hospedar) vai enxergar os mesmos dados, em tempo real, de qualquer dispositivo.
